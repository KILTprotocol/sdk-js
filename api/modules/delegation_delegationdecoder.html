External module Delegation/DelegationDecoder

When DelegationNodes or DelegationRootNodes are written on the blockchain, they're encoded. DelegationDecoder helps to decode them when they're queried from the chain.

The DelegationDecoder methods transform a QueryResult into an object of a KILT type.
